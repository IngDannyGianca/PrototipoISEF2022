Guardar Nómina
Para guardar la información generada en la base de datos se debe presionar el botón
y al finalizar el proceso se visualizará una ventana como la siguiente:
Imprimir Nómina
Al finalizar el guardado se puede imprimir la tabla generada presionando el botón , lo
cual nos permite también guardar el documento con extensión PDF, siguiendo los pasos a
continuación.
Se debe seleccionar la opción Microsoft Print to PDF, como se ve visualiza en la siguiente
imagen:
Luego se debe seleccionar el botón y se abrirá el explorador de archivos.
Se debe ingresar un nombre para el nuevo archivo y presionar el botón y se
generará un archivo en la ruta seleccionada.
Si se abre el archivo se verá de la siguiente manera:
Se indica la fecha de creación del reporte, el número de nómina, el inicio y final del periodo, el
total de percepciones, deducciones y total líquido.
